## <span id="member">Memberコンポーネント</span>

メンバーとそのソーシャルリンクを列挙します。

<Member members={[
	{
		name: 'Rinrin.rs',
		socials: {
			twitter: 'Rinrin_2nd',
			youtube: '@rinrin-rs',
			twitch: 'rinrin0413',
			other: 'https://rinrin.pages.dev'
		}
	},
	{
		name: 'REVATI',
		socials: {
			twitter: 'revati_jp',
			youtube: '@revati_jp',
			others: 'https://revati.jp'
		
		}
	},
	{
		name: 'しろねこ',
		socials: { twitter: '46nek_0' }
	},
	{ name: 'あるてぃめっと名無し!' }
]} />

membersプロパティで受け取れる型は次の通り:

```
{
    name: string;
    socials?: {
        twitter?: string;
        youtube?: string;
        twitch?: string;
		other?: string;
    };
}[]
```
